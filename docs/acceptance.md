# V1 验收清单

## 1) 功能验收

- [ ] 对话：连续 20 轮无崩溃；平均响应 < 2.5s（网络正常）
- [ ] 人格一致性：30 条 prompt 评分 >= 80/100
- [ ] 待办：基础 CRUD 成功率 100%
- [ ] 定时触发：提醒误差 <= 60s
- [ ] 自言自语：开启 2 小时内不超过上限；关闭后 0 次
- [ ] 电脑操作：白名单动作成功率 >= 95%
- [ ] 高风险动作未确认执行率 = 0%

## 2) 安全与隐私

- [ ] 未开启系统控制时，所有 system_action 被拒绝
- [ ] 所有系统动作均写入审计日志
- [ ] 不提供“一键清空数据”入口或命令

## 3) 性能稳定性

- [ ] 冷启动到可交互 < 3s
- [ ] 空闲内存 < 250MB
- [ ] 24h 连续运行无主进程崩溃

## 4) 可用性

- [ ] 10 分钟新手路径：首次对话 + 创建待办 + 收提醒
- [ ] 核心开关 2 次点击内可达
- [ ] 失败场景提示清晰（网络失败、权限、动作拒绝）
